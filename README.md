📝 Responsive HTML Registration Form

A responsive and visually appealing registration form built using HTML and CSS.
This project demonstrates form structure, input handling, styling techniques, and background image integration.

🚀 Project Overview

The Responsive HTML Registration Form is designed to collect user details in a structured and user-friendly format.
The form is centered on the page with a background image (flo.jpg) and includes various input elements such as text fields, radio buttons, checkboxes, dropdowns, and a submit button.

🛠️ Technologies Used

*HTML5
*CSS3 (Internal Styling)

🎨 Features

✔ Full-screen background image (flo.jpg)
✔ Centered form using Flexbox
✔ Soft pastel form background with opacity
✔ Rounded corners and box shadow
✔ Hover animation on Submit button
✔ Responsive layout
✔ Multiple input types:

*Text
*Email
*Password
*Date
*Radio buttons
*Checkboxes
*Dropdown menu
*Textarea
